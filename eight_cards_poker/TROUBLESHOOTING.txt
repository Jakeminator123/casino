TROUBLESHOOTING - Eight Cards Poker v2.0
==========================================

If the new features (winner hands display, mixed PLO/NL) aren't working:

1. CLEAR BROWSER CACHE (IMPORTANT!)
   - Press Ctrl+Shift+Delete in your browser
   - OR press F12 -> Right-click refresh button -> "Empty Cache and Hard Reload"
   - OR open the game in an Incognito/Private window

2. RESTART THE SERVER
   - Stop the current server (Ctrl+C)
   - Run START_GAME.bat again

3. VERIFY VERSION
   - Open browser console (F12)
   - You should see: "Eight Cards Poker v2.0 - Enhanced with winner hands display"
   - The page title should show "Eight Cards Poker - Multiplayer v2.0"

4. CHECK CONSOLE FOR DATA
   When showdown happens, check browser console (F12) for:
   - "Showdown results received:" followed by data
   - "ShowFinalResults data:" followed by data
   - Check if "winner_hand", "p1_type", "p2_type" fields are present

5. IF USING NGROK
   - Make sure both players refresh their browsers (Ctrl+F5)
   - Try a new ngrok session if issues persist

NEW FEATURES IN v2.0:
- Shows what hand won on each board (e.g., "Player 1 Wins with Flush!")
- Players can choose different boards for PLO (e.g., P1 plays PLO on Board A, P2 plays PLO on Board B)
- Mixed game display shows "P1: PLO vs P2: NL" when applicable
